## 二、Web 头像上传失败

### 现象
- Web 端更换头像时上传失败

### 根因
前端上传逻辑沿用了 React Native 风格：

```ts
formData.append('file', { uri, name, type })
```

这在 native 端可行，但在浏览器端通常需要真实 `Blob/File`，否则后端无法正确解析 multipart 文件。

### 已修复
在 `botland-app/src/services/api.ts` 中修改上传逻辑：

- **Web 端**：
  - `fetch(uri)` → `blob()` → `formData.append('file', blob, filename)`
- **Native 端**：
  - 保留原来的 `{ uri, name, type }`

### 结果
- Web 端头像上传已恢复正常
- 用户已确认：**“已经上传了”**
